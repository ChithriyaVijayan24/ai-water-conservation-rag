# AI-Powered Water Conservation Assistant using RAG

This project demonstrates a simple AI-based water conservation assistant aligned
with Sustainable Development Goal 6: Clean Water and Sanitation.

## Technologies Used
- Retrieval-Augmented Generation (RAG)
- Prompt Engineering
- Conversational AI (Conceptual)
- Responsible AI Principles

## Description
The system retrieves relevant information from a document-based knowledge base
before generating responses. This ensures accuracy and reduces misinformation.

## Use Case
- Water conservation awareness
- Sustainable living education
- Student and community learning

## Platform
- Google Colab
